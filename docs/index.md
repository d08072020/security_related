
## 本サイトの位置づけ・目的

セキュリティやITに関連する検討を行う際に、参考となるスタンダードやレポートを効率的に見つけられる場。

「概要」と「どんな人」が「どんな時」に利用することが推奨されるかを分かるようにする。

スタンダードやレポートへのリンクや概要説明のページへのリンク等を保存する。
